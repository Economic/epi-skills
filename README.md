# epi-skills

A Claude Code plugin marketplace for economics and data analysis. We use these skills at the Ec

## Available plugins

| Plugin | Description |
|--------|-------------|
| [r-econ-data](plugins/r-econ-data/) | Economic data sources and retrieval methods for R (BEA, BLS, Census, EPI CPS extracts, FRED, IPUMS, SWADL) |

## Installation

Add the marketplace:

```
/plugin marketplace add economic/epi-skills
```

Then install a plugin:

```
/plugin install r-econ-data@epi-skills
```
